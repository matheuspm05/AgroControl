# Regras de Negócio

- Um usuário possui uma fazenda.
- Uma fazenda possui animais, pastos, currais, remédios e campeiros.
- Um animal pertence a uma fazenda.
- Um animal não pode estar em dois locais ao mesmo tempo.
- Um curral possui capacidade máxima.
- O sistema deve impedir movimentação para curral lotado.
- Toda aplicação de remédio deve ficar registrada no histórico do animal.
- Toda movimentação de animal deve ficar registrada no histórico.
- Um usuário só pode acessar os dados da própria fazenda.
- Pastos e currais pertencem a uma fazenda.
