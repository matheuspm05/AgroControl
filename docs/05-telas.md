# Telas do Sistema

## Públicas

### Login
Tela para entrada no sistema.

Campos:
- Email
- Senha

Ações:
- Entrar
- Ir para cadastro

### Cadastro
Tela para criar conta.

Campos:
- Nome
- Email
- Senha
- Confirmar senha
- Contato

Ações:
- Cadastrar
- Ir para login

## Privadas

### Dashboard
Tela principal após login.

Informações:
- Total de animais
- Total de pastos
- Total de currais
- Total de remédios
- Últimas movimentações

### Minha Fazenda
Informações gerais da fazenda.

### Animais
Lista de animais cadastrados.

Ações:
- Cadastrar animal
- Editar animal
- Excluir animal
- Ver detalhes

### Pastos
Lista de pastos cadastrados.

### Currais
Lista de currais cadastrados.

### Remédios
Lista de remédios cadastrados.

### Campeiros
Lista de campeiros cadastrados.

### Movimentações
Tela para movimentar animal entre locais.

### Aplicações de Remédio
Tela para aplicar remédio em animal.

### Relatórios
Tela com resumos e informações da fazenda.
