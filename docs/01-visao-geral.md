# Visão Geral do Projeto

## Nome do projeto

Gerenciador de Fazenda Web

## Descrição

Sistema web para gerenciamento de uma fazenda, permitindo cadastrar e organizar animais, pastos, currais, remédios e campeiros, além de movimentar animais e visualizar informações da estrutura da fazenda.

## Problema que resolve

O sistema centraliza o gerenciamento da fazenda em um único lugar, facilitando o controle dos animais e da estrutura da propriedade.

## Público-alvo

- Fazendeiros
- Pequenos e médios produtores rurais
- Pessoas que precisam organizar o rebanho e a estrutura da fazenda

## Objetivo principal

Criar uma versão web do sistema originalmente desenvolvido em Java console, agora com interface moderna, persistência de dados e possibilidade de evolução futura.
